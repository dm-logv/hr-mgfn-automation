# hr-mgfn-automation

Решения к тестовым заданиям Мегафон на позицию автоматизации процессов.


## Структура проекта

* `gppl` -- Задания по языкам общего назначения
    * `go` -- Go
    * `js` -- JavaScript
    * `py` -- Python
* `sql` -- Задания по SQL


## Запуск

### JS

```sh
cd hr-mgfn-automation/gppl/js
./run
```
